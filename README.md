## GIT BASH😁

1. **git config** --list = comando que lista as configurações do git bash.

2. **git config --global user.name** = para setar o nome.

3. **git config --global user.email** = para setar o email.

4. **git config --global user.password** = para setar a senha.

5. **git config --global --unset--all user** = oque vc quer tirar. Ex name, email, etc.

## TERMINAL😎

1. **cls** = limpa cmd

2. **cd** = escolhe pasta

3. **cd ..** = volta pasta

4. **mkdir** = cria diretório

5. **rmdir** = apaga pasta vazia

6. **tree** = busca subpastas

7. **edit** = pode editar arquivos executáveis(tipo um texto)

8. **cd/** = volta pro primeiro disco

9. **explorer** = abre pasta no windows
    

## PARA O GIT HUB(TERMINAL)😺
   
10. **git clone** (link)

11. **git status**

12. **git add** (nome do arquivo)

13. **git commit -m** (descrição)

14. **git push** (linka no github)

15. **git pull** ( linka do git para o pc)
